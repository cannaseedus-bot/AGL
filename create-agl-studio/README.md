# create-agl-studio

A minimal Vite + React template for booting AGL Studio instantly.

## Usage

```
npm create agl-studio
```

Then:

```
cd my-app
npm install
npm run dev
```

This launches the full AGL Studio:

- glyph palette
- graph canvas
- drag-and-drop
- execution panel
- model-assisted flow generator
